Jobby App

A modern job search web application built using React that allows users to browse jobs, view job details, and filter jobs based on employment type and salary range.

This project demonstrates my skills in React, routing, API integration, authentication, and responsive UI development.

🚀 Live Features

🔐 Login Authentication

📄 View Job Listings

🔎 Search Jobs

🏷️ Filter by Employment Type

💰 Filter by Salary Range

📑 Detailed Job Description Page

❌ Error Handling & Retry Option

🔒 Protected Routes

📱 Responsive Design
